# Canvas Design Skill

Create visual art through a two-phase process: aesthetic philosophy → canvas expression.

## Phase 1 — Design Philosophy

Develop a named visual movement (4–6 paragraphs) articulating how ideas manifest through space, form, color, and composition — not text. Emphasise meticulously crafted work that appears to have taken countless hours by someone at the absolute top of their field.

Output as `.md` file.

## Phase 2 — Canvas Expression

Translate philosophy into a single-page PDF or PNG that is:
- **90% visual design, 10% essential text**
- Museum-quality — not decorative, not clip-art
- Typography as visual element, not communication vehicle
- Every element with breathing room — nothing overlaps
- Proper margins on all sides

## Key Principles

- Information lives in visual design, not paragraphs
- Sparse, clinical typography — text anchors, doesn't explain
- Embed subtle conceptual references that reward close viewing
- Refine existing elements rather than adding new ones
- Sophistication regardless of subject matter — never cartoony

## Output Formats

- `.md` — design philosophy
- `.pdf` or `.png` — visual expression

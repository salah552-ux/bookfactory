## AMAZON KINDLE SERIES PAGE — Cathedral Close Mysteries

### How to set up a Kindle series page

Amazon requires the series to be registered. Steps:

1. Go to **KDP → Bookshelf → Death in the Cathedral Close → Book Details**
2. Scroll to **Series information**
3. Set **Series title:** The Cathedral Close Mysteries
4. Set **Volume:** 1
5. Save changes

This creates a Kindle series page. When Book 2 is uploaded with the same series name and Volume 2, Amazon links them automatically. Readers see both books on the series page.

**Note:** The series page does not appear until at least 2 books in the series are published. Setting it now is correct — it pre-registers the series name and volume so Book 2 links automatically when published.

### Goodreads series page

1. Log into Goodreads (create account if needed — use author identity)
2. Go to the Goodreads page for Death in the Cathedral Close
3. Click "Edit book" → find "Series" field
4. Add: **The Cathedral Close Mysteries #1**
5. This creates a series page on Goodreads automatically
